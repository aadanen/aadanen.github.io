Why did I do it?

I'm currently researching under Dr. Olivier Hervet to improve Bjet_MCMC because I'm interested in working with computationally demanding programs, and I think Astrophysics as well as many other physical sciences are a really interesting place to apply computer science. It's exciting for me to work on projects that are pushing the boundaries and demand the highest quality of work.

What did I accomplish?

I've been using the Hummingbird Computer Cluster to test Bjet and gather data for the experiment I designed. I've written a bunch of python to automate tasks such as extracting and interpreting Bjet's output and creating graphs. I've also modified existing python code to expose previously hidden details about the proccess. I then took my results, prepared a presentation, and shared my findings with the UCSC Very High Energy Astrophysics Department.

What did I learn?

The most important thing I'm currently learning is how to deal with a large, complex project, that I didn't write myself. Most of my programming at school is individual and small scale, so working to augment and improve an existing codebase is a fun experience. I've also gotten to practice larger scale software developement, and I have been solving problems like deciding how I am going to organize my code.
